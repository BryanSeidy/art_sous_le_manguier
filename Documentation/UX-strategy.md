🌿 STRATÉGIE UX/UI DESIGN
Projet : Art Sous le Manguier — Expérience Web Immersive
1. Vision Produit & Positionnement
🎯 Objectif principal

Transformer Art Sous le Manguier en une expérience digitale immersive, sensorielle et narrative, dépassant le simple site vitrine pour devenir :

Un lieu numérique vivant
Un espace culturel interactif
Une extension digitale de l’arbre à palabres
💡 Positionnement UX

“Ne pas naviguer dans un site, mais entrer sous un manguier.”

Le produit se situe à l’intersection de :

🎨 plateforme artistique
🎧 média culturel immersif
🌍 expérience interactive narrative
2. ADN Conceptuel : Le Manguier comme Système UX
🌳 Métaphore Fondatrice

Le manguier devient l’architecture UX principale :

Élément	Fonction UX	Traduction Interface
Racines	Origines / histoire	About / Vision / Archives
Tronc	Contenu principal	Accueil / Live / Épisode du moment
Branches	Navigation	Catégories artistiques
Feuilles	Contenu secondaire	Articles / extraits
Fruits	Contenu premium	Épisodes / œuvres clés
🔁 Paradigme UX
Navigation non-linéaire
Exploration organique
Interaction intuitive & sensorielle
3. Personas & Intentions Utilisateurs
🎭 Persona 1 : Le Passionné d’Art
Cherche : découverte, inspiration, immersion
Attentes : expérience visuelle forte, contenu riche
Friction : sites trop classiques, manque d’émotion
🎤 Persona 2 : L’Artiste / Invité
Cherche : visibilité, storytelling
Attentes : mise en valeur élégante de son travail
Friction : présentation générique, manque de narration
🌍 Persona 3 : Le Curieux Digital
Cherche : expérience innovante
Attentes : interactions surprenantes, design moderne
Friction : UX ennuyeuse
4. User Journey Immersif (Parcours Émotionnel)
🌑 1. L’Éveil (Landing Experience)
Écran sombre, ambiance sonore subtile
Apparition progressive du manguier
Interaction : mouvement souris = lumière

👉 Objectif : captiver immédiatement

🌿 2. L’Exploration (Navigation organique)
Zoom dans l’arbre
Objets interactifs :
🍃 feuille → archives
🥭 fruit → épisode
🎨 trace → galerie

👉 Objectif : remplacer menu classique par exploration

🎬 3. L’Immersion (Page contenu)
Scrollytelling narratif
Décomposition visuelle (image → œuvre → détail)
Audio contextuel

👉 Objectif : créer une expérience émotionnelle forte

🗣️ 4. Les Palabres (Communauté)
Commentaires intégrés dans décor
Interaction sociale poétique

👉 Objectif : engagement communautaire

5. Direction Artistique (UI Design System)
🎨 Concept visuel : Ethno-Futurisme

Fusion :

Tradition africaine 🌍
Design contemporain ✨
Technologie immersive 💻
🎨 Palette de couleurs
Couleur	Usage	Code

Jaune mangue	Accent	#F4A261
Terre de Sienne	Texture	#8B4513
Blanc cassé	Texte	#F4F1DE
Indigo profond	Contraste	#1D3557
🔤 Typographie
Titre : Serif artistique (Playfair Display)
Corps : Sans-serif moderne (Montserrat / Space Grotesk)

👉 Dualité : culture vs modernité

🧩 UI Components Signature
Curseur = 🎨 pinceau / halo interactif
Boutons = formes organiques
Images = masques irréguliers (feuilles / peinture)
Cards = flottantes avec inertie

7. Architecture Technique (Frontend)
⚙️ Stack recommandée
Framework : Next.js (App Router)
Animation : Framer Motion + GSAP
3D / WebGL : Three.js
Shaders : GLSL
Audio : Web Audio API
🚀 Optimisation
Mobile-first impératif
Lazy loading immersif
GPU acceleration
fallback version simplifiée mobile
8. Mobile Experience (Crucial)

Simplification WebGL

👉 priorité : fluidité > complexité
